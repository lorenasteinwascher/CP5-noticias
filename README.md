## Funcionalidades
- Lista ≥10 notícias (array estático).
- Busca por **Título, Conteúdo e Categorias**.
- Página da notícia com **comentários**.
- Botão “Voltar” e **404 customizada**.
- Layout **responsivo** e CSS externo.
